Resources

This page contains a list of resources that can be used to understand the principles of data systems for education and how to use data effectively.

Case Studies

A set of case studies that can be used to understand how data is used different education quality improvement programmes in India.

  • Case Study: The Role of Data in an Online Teacher Professional Development Programme – A case study on how data was used in an online teacher professional development programme of over one lakh teachers in Rajasthan.

  • Case Study: Using Performance Data to Enhance the Robustness in Government Systems and Data-based Decision Making – A case study on how data was used in a programme to improve learning outcomes in Uttar Pradesh.

  • Case Study: Assessment Data to inform Improvement in Learning Outcomes – A case study on how data was used in a programme to improve learning outcomes through a Graded Assessment test in Arunachal Pradesh.

Checklists

A set of checklists to help you navigate your data journey.

  • Checklist for Data Collection – A checklist to help you navigate the data collection process in a more robust manner.

  • Checklist for Data Entry with Google Sheets – A checklist to ensure that data is entered in a consistent manner even when multiple people are entering data.

Other Resources

  • Education Datasets Wiki - A primer on the datasets that are available for education in India.
  • Principles and Good Practices - A set of principles and good practices to follow while conceptualising a data system for education.
  • Taking a Macroscopic Perspective of Data Management - A short note to help you understand the principles of data management at a systems level.
